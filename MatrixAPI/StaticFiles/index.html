Get started with Bootstrap

The application will consist of two pages. One [Input page] for data entry, second [Output page] will display the result. User will set dimensions of both matrixes in range from 1 to 5 and then supply input values. User clicks “=” button which submits request to the calculator web service (B), waits for a result and the result will be displayed on the Output page.
